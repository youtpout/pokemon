I. AVERTISSEMENT 
------------------------------------------------------------------------------------------------------------
Cette version est une preview.

Pour le moment sur un projet Windows 8.1 UAP ou un projet Windows 10 UWP 2 dll s'installent :

- Navegar.UAP.Win81
- Navegar.UWP.Win10

Il faut supprimer la dll ne correspondant pas à la version du framework que vous ciblez.
